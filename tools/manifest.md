# Tools Manifest

> Master list of available tools and their functions.

## Memory Tools (starter placeholders)
- `tools/memory/memory_read.py` — Read `memory/MEMORY.md` and daily logs; output in requested format.
- `tools/memory/memory_write.py` — Append events/facts to logs or update `memory/MEMORY.md`.
- `tools/memory/memory_db.py` — Keyword search in `data/memory.db`.
- `tools/memory/semantic_search.py` — Semantic search over memory entries.
- `tools/memory/hybrid_search.py` — Hybrid keyword + semantic search.
